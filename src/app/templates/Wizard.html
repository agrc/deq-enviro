Wizard

If this is your first time using this application, it is strongly recommended that you begin with this wizard. If you have experience with this application, you can close this pane.

Information from the Drinking Water Source Protection program (Utah Division of Drinking Water) is available by requesting access. The source protection information includes public water sources, ground-water protection zones, surface-water protection zones, and source water assessment zones.

Step 1: Select Layers to Search

This step allows you to select specific data layers to search. Your search results will only contain/display data from the layers you have selected.

There are custom filters for some layers that will appear when their corresponding layers are selected.

Step 2: Enter Search Values and Execute Search

This step allows you to search the data in different ways, but you can ONLY USE one search method at a time.

Search by Address allows you to search the data based on a buffer around an address.

Search by “City” or “County or Statewide” allows you to search all the data in a specific city, county or the entire state.

Search by Site Name allows you to search by the name of the facility, property, water system or other identifier.

Search by Program ID allows you to search for program specific identification numbers.

Search by Shape allows you to draw your own custom shape to search the data. As the cursor is placed over each shape tool, a brief description of its function appears.

Step 3: Analyze Search Results

This step allows you to review the results of your search. Using the tools in the right sidebar you can start a new search, download results, clear graphics, modify search, or export map images. You can also select individual records to:

  • View specific site information
  • Submit an electronic request to review the file(s) by selecting the “GRAMA Request” link.
  • View electronic documents associated with the specific site.

To select a subset of results for download click on the rows on the grid while holding down the control/command button.

PLEASE NOTE: When viewing the search results "n/a" means that the data is currently not available online.
